# BashRepeatCommand
Repeates given command x times

## How to use

```bash
repeat $COMMAND $NUMBER OF TIMES
```

For example:

```bash
repeat neofetch 10
```

For commands with spaces, use ""

```bash
repeat "neofetch --arch_linux" 10
```

## Installation

1. Download script

2. Give permissions

```bash
sudo chmod +x repeat
```

3. Move to /bin/

```bash
sudo mv repeat /bin/repeat
```
